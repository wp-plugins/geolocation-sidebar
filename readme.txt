== Description ==

This widget shows visitor's location on map.

== Installation ==

1. Extract the zip file geolocation-sidebar.zip, then you will see the directory which is named as geolocation-sidebar

2. Upload the geolocation-sidebar file to the `/wp-content/plugins/` directory on the your web server

3. Activate the plugin through the 'Plugins' menu in WordPress admin panel

4. Navigate to Appearance -> Widgets and then you will see Geolocation Sidebar widget, drag and drop it to sidebar

5. Now, you can type the Title that is shown on your site, by default width and height of map image are 200px, you can change this values.

== Frequently Asked Questions ==

= How do this get the location data? =

This plugin uses the maxmind open source API, all geolocation data is in GeoLiteCity.dat

NOTE: You should update this file monthly from maxmind

Direct URL :

http://geolite.maxmind.com/download/geoip/database/GeoLiteCity.dat.gz


Maxmind :

http://www.maxmind.com/app/geolitecity


After downloading process, extract the .dat file and then delete old database, upload this database instead into the `wp-content/plugins/Geolocation_sidebar`
